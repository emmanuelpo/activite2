Activite 2: Git et GitHub

Cette activité nous permet de prendre en main et d'apprender à utilise Git et GitHub
Ici, nous devons créer un repository sur notre compte GitHub puis nous devons créer trois fichiers distincts.
L'un de ces fichiers se nomme README.txt et les deux autres sont au choix.
De plus, nous devons faire 4 commits (modification ou ajout de fichier) distincts.
